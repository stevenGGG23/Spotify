A simple web app that scans your Spotify playlist and uses Spotify's own recommendation API to generate a new playlist with similar songs you don't already have.

## What it does

- Log in with your Spotify account
- View your playlists
- Choose a playlist to analyze
- Automatically get recommended tracks based on that playlist
- Create a new Spotify playlist with those recommendations

## Tech Stack

- Frontend & Backend: Next.js (React)
- APIs: Spotify Web API
- Hosting: Vercel
